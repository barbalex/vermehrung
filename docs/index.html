Bedrohte Pflanzen vermehren

Arten

Vom Aussterben bedrohte Pflanzen-Arten werden von Experten gezielt vermehrt und wieder ausgesetzt.

Herkünfte

...sind natürliche Vorkommen. Hier werden die Arten gesammelt.

Sammlungen

...beschreiben das Sammeln von Arten an einem Herkunfts-Ort.

Lieferungen

Sammlungen werden an Kulturen geliefert. Aus Kulturen werden Pflanzen zur Auswilderung geliefert. Oder an andere Kulturen. Lieferungen können geplant und bestellt werden.

Gärten

In Gärten und Gärtnereien werden die Arten vermehrt und auf das Aussetzen vorbereitet.

Kulturen

Die Pflanzen einer Art in einem Garten bilden eine Kultur.

Events

...beschreiben Ereignisse im Rahmen der Kultur.

Inventare

In grösseren Gärtnereien wird über Kulturen und deren Beete ein Inventar geführt.

Zählungen

Regelmässig wird der Bestand einer Kultur erfasst. So können Sammlungen und Aussetzungen geplant werden
