Progressive Web App

21.01.2019

Wieso kann vermehrung.apflora.ch auf allen Geräten und Betriebssystemen gleichermassen verwendet werden?

Weil es eine Progressive Web App ist (PWA).

Progressive Web Apps sind Web-Applikationen und werden für Browser entwickelt.

Weil sie aber einige zusätzliche Anforderungen erfüllen, können sie je länger je weniger von "normalen" Apps unterschieden werden. Google und Microsoft arbeiten zügig daran, dieses Ziel zu erreichen.

Was ist der Vorteil gegenüber einer "normalen" App?

  • Die App funktioniert auf allen Bertriebssystemen und auf allen Geräten mit modernem Browser
  • Sie finden die App im Internet (und auf Windows künftig im App-Store)
  • Sie müssen die App nicht installieren. Das geschieht automatisch, wenn Sie die URL besuchen
  • Updates erfolgen automatisch
  • Einer neuen Mitarbeiterin schicken Sie einfach Link und Login. Schon kann sie loslegen!

Je nach Betriebssystem und Browser werden Sie irgendwann gefragt, ob Sie vermehrung.apflora.ch auf dem Desktop ablegen möchten (oder Sie können es selber veranlassen). Danach öffnet sich vermehrung.apflora.ch ohne die URL-Zeile und andere Teile des Browsers. Und ist fortan auch optisch kaum mehr von "normalen" Apps unterscheidbar.

Ist Ihnen aufgefallen, dass Apple oben nicht zusammen mit Google und Microsoft erwähnt wurde?

Apple verdient sein Geld mit dem App-Store. 30% der App-Erlöse gehen an Apple. Bei PWA's gehen 100% des Erlöses an die Entwickler. Daher führt Apple die für Progressive Web Apps nützlichen Technologien auffällig langsam ein. Und darum funktionieren Progressive Web Apps auf Android und Windows besser als auf MacOS und iOS. Trotzdem funktionieren sie auch auf Apple-Geräten. Sie sind eben auch eine Web-App.
